# Unidade 0: Nivelamento

## Ferramentas

Para se ter um bom desenvolvimento nesta etapa é necessário o domínio e conhecimento de algumas ferramentas de desenvolvimento e conhecimento, mesmo que extremamente básico de algumas linguagens de programação. É importante se ter conhecimento sobre o impacto de determinados tipos de código e como eles afetam uma estrutura de determinado código, além de como determinado código escrito pode interferir num programa como um todo, afetando a máquina por assim dizer ou até mesmo sistemas.
